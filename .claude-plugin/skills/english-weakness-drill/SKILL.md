# English Weakness Drill

한국어 문장을 보고 영어로 즉시 말하게 하는 반사 드릴로 문법 약점을 훈련한다. 틀린 패턴만 골라 변형 재출제하고, 끝나면 오답노트를 자동 저장한다.

## Quick start

약점 로드 → 훈련 방식 안내 → 라운드 출제 → 즉시 교정 → 적응형 반복 → 오답노트 저장.

## Step 1 · 약점 로드

이 스킬은 `english-conversation-review`와 한 세트다. review가 회화록을 분석해 약점을 `06_English/Review/`에 남기면, drill이 그걸 읽어 훈련한다. 두 곳에서 약점을 가져온다:

1. **`06_English/Review/history.json`** (Obsidian vault: `/Users/jayden/Obsidian-vault/06_English/Review/history.json`) — review가 누적한 공유 메모리. 각 세션 `mistakes` 태그를 모아, **2회 이상 반복된 태그**를 최우선 약점으로 잡는다 (만성 약점부터 훈련).
2. **`06_English/Review/`의 최신 오답노트(.md)** — 구체적인 틀린 문장 예시를 패턴 변형의 참고로 쓴다.
3. 위 둘 다 없으면 사용자에게 약점 요약/플래시카드를 붙여넣어 달라고 요청한다.
4. 이전 오답 문장을 그대로 재출제하지 않는다. 약점 "패턴"만 가져와 새 변형 문제를 만든다 (망각곡선 복습 아님).

## Step 2 · 훈련 방식 안내

사용자에게 3단계 적응형 루프를 먼저 설명한다:
- 1단계 진단 드릴: 약점 섞은 한→영 문제. 떠오르는 대로 말하고 타이핑.
- 2단계 즉시 교정: 틀린 부분만 콕 집어 ①왜 틀렸는지 ②규칙 한 줄 ③교정문.
- 3단계 적응형 반복: 틀린 패턴만 변형 재출제. 2연속 정답이면 졸업, 막히면 더 쉽게 쪼갬.
- 원칙: 정답을 미리 보여주지 않는다. 사용자가 먼저 생산(production)해야 입에 붙는다. 막히면 "힌트" 요청.

## Step 3 · 라운드 출제

- 표 형식으로 낸다: `| # | 타겟 약점 | 🇰🇷 한국어 |`.
- 한 라운드 5~8문제. 약점을 골고루 섞는다.
- 플래시카드/노트 예문을 그대로 베끼지 말고 상황을 바꿔 변형 출제한다 (외운 것 말고 실제 아는지 확인).
- 사용자는 번호 붙여 답한다. 막힌 번호는 "모름"으로 둔다.

## Step 4 · 즉시 교정

- 잘한 것 먼저 짚는다 (어떤 함정을 피했는지).
- 틀린 것만 교정한다. 각 항목은 ❌/✅/👉 3줄 템플릿 (아래 동일).
- 한 문장에 오류 여러 개면 모두 분리해 설명한다.

## Step 5 · 적응형 반복

- 패턴 점수표를 보여준다 (정답/오답/미응답, 졸업 여부).
- 틀린 패턴만 같은 함정 다른 상황으로 변형 재출제한다.
- 2연속 정답이면 "졸업". 계속 막히면 더 쉬운 버전으로 쪼갠다.
- 모든 약점 패턴이 졸업할 때까지 라운드를 반복한다.

## Step 6 · 오답노트 저장

드릴이 끝나면 `06_English/Review/`에 오답노트를 자동 저장한다. 형식·파일명·문체 규칙은 [REFERENCE.md](REFERENCE.md)를 따른다.

## 오답 항목 템플릿 (드릴 교정 + 오답노트 공통)

```
❌ I had a training last month.
✅ I had safety training last month.
👉 training은 불가산 명사 → 관사 a 못 붙임.
```
